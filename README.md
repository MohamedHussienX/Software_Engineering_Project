# Software_Engineering_Project-
What the project does:

This website helps users manage and keep track of engineering equipment like tools, machines, and consumables. Users can:
Add new equipment to the inventory.
Update details (like quantity or status) of existing equipment.
Search and filter equipment by category or supplier.
View all items available in the inventory.
Delete old or broken equipment.
Why the project is useful:

The system saves time by organizing all equipment in one place and making it easy to search for what you need. It helps prevent errors like losing track of items or over-purchasing supplies.
How users can get started with the project:

Log in to the system with a username and password.
Navigate the simple interface to view, search, or update the inventory.
If you're an admin, you can also add or remove equipment, update details, or manage users.
